# 🔧 聊天频道分离修复

## 📅 修复日期
2026-03-07 12:26

## 🐛 问题描述

**用户反馈：**
> 新版本为什么聊天页签，我切换飞书、和 webui 是一样的内容，飞书不是应该只显示飞书通道的消息吗

**问题原因：**
1. 前端组件中，所有频道（WebUI、飞书、全部）都调用相同的 `loadMessages()` 方法
2. 没有为不同频道指定对应的会话 Key
3. 导致切换频道后加载的是同一个会话的消息

---

## ✅ 修复方案

### 1. 频道配置添加 sessionKey

```javascript
const CHANNELS = [
  { 
    id: 'webui', 
    name: 'WebUI 会话', 
    icon: '🌐', 
    color: 'text-green-400', 
    sessionKey: 'agent:main:main' // 主会话
  },
  { 
    id: 'feishu', 
    name: '飞书', 
    icon: '📱', 
    color: 'text-blue-400', 
    sessionKey: 'agent:main:feishu:group:oc_2ea57de39e1f2f14bf549cb79075168f' // 飞书群聊
  },
  { 
    id: 'all', 
    name: '全部', 
    icon: '📋', 
    color: 'text-gray-400', 
    sessionKey: null // 全部频道显示所有会话
  },
];
```

### 2. 切换频道时加载对应会话

```javascript
useEffect(() => {
  const channel = CHANNELS.find(c => c.id === selectedChannel);
  
  if (channel?.sessionKey) {
    // 有指定会话的频道（WebUI、飞书）
    loadSessionMessages(channel.sessionKey);
  } else if (selectedChannel === 'all') {
    // 全部频道：加载所有会话
    loadSessions();
  } else {
    // 其他频道
    loadMessages();
  }
}, [selectedChannel]);
```

### 3. 发送消息到对应频道

```javascript
const sendMessage = async () => {
  const channel = CHANNELS.find(c => c.id === selectedChannel);
  const targetSessionKey = channel?.sessionKey;
  
  if (targetSessionKey) {
    // 发送到指定会话（WebUI 或飞书）
    await fetch('/api/chat/sessions/send', {
      method: 'POST',
      body: JSON.stringify({
        sessionKey: targetSessionKey,
        message: inputMessage,
      }),
    });
  } else {
    // 全部频道：默认发送到主会话
    await fetch('/api/chat/sessions/send', {
      method: 'POST',
      body: JSON.stringify({
        sessionKey: 'agent:main:main',
        message: inputMessage,
      }),
    });
  }
};
```

### 4. UI 优化

移除会话选择器，显示当前频道信息：

```jsx
{/* 当前频道信息 */}
{selectedChannel !== 'all' && (
  <div className="text-xs text-gray-400">
    📍 当前：<span className="text-brand font-medium">
      {CHANNELS.find(c => c.id === selectedChannel)?.name}
    </span>
    {wsConnected && <span className="ml-2 text-green-400">✓ 实时同步</span>}
  </div>
)}
```

## 🚀 使用说明

### WebUI 会话 🌐
- **显示**：你与 AI 的直接对话（主会话）
- **发送**：消息发送到主会话
- **场景**：个人对话、任务执行、代码编写

### 飞书 📱
- **显示**：飞书群聊中的消息（机器人@互动）
- **发送**：消息发送到飞书群聊
- **场景**：群聊协作、团队通知

### 全部 📋
- **显示**：所有会话列表
- **发送**：默认发送到主会话
- **场景**：查看有哪些会话
