# CHANGELOG v2.0.3

## 发布日期
2025-03-07

## 概述
本次更新修复了 Dashboard v2.0.3 的 10 个核心问题，提升了用户体验和系统稳定性。

## 修复内容

### 1. 当前任务模块显示优化 ✅
**问题：** `openclaw-cn task current` 返回空，但实际有 subagents 在运行

**修复：**
- 修改后端 API 从 `openclaw-cn subagents list` 获取真实任务
- 显示当前正在执行的 subagent 任务
- 无任务时显示友好提示"系统空闲，等待指令"

**文件：** `server/routes/tasks.js`

---

### 2. 移除运行时长显示 ✅
**问题：** `ps -o etime` 获取失败，运行时长显示 0s

**修复：**
- 删除 StatusBar 中的"运行时长"字段
- 移除 `formatUptime` 函数
- 简化状态栏显示

**文件：** `src/components/StatusBar.jsx`

---

### 3. ToDo 列表优化 ✅
**问题：** ToDo 列表一直是空的，数据来源不明

**修复：**
- 确认数据存储在 localStorage (key: 'dashboard-todos')
- 添加空状态友好提示"✨ 暂无待办"
- 添加数据来源说明：浏览器本地存储，清除缓存会丢失
- 保留手动刷新功能

**文件：** `src/components/TodoList.jsx`

---

### 4. Cron 命令验证正则修复 ✅
**问题：** 正则太严格，"备份每日记忆文件"包含中文无法通过验证

**修复：**
- 修改验证正则：`/^[\u4e00-\u9fa5a-zA-Z0-9_\-\s]+$/`
- 允许中文、字母、数字、下划线、连字符、空格
- 更新错误提示信息

**文件：** `src/components/CronManager.jsx`

---

### 5. 预设 Cron 命令修正 ✅
**问题：** 部分预设命令格式不正确或不可执行

**修复：**
- 修正"记忆备份"命令：添加 `mkdir -p` 确保目录存在
- 修正"日志清理"命令：添加 `-type f` 仅删除文件
- 移除"心跳检查"命令（`openclaw-cn heartbeat` 可能不存在）
- 添加"系统诊断"命令（`openclaw-cn doctor`）
- 更新 Cron 预设标签："午夜" → "0 点"

**文件：** `server/routes/cron.js`

---

### 6. Cron 表达式帮助 ✅
**问题：** 用户不熟悉 Cron 表达式格式

**修复：**
- 将"执行时间（午夜）"改为"执行时间（0 点）"
- 在自定义 cron 表达式下方添加 ❓ 按钮
- 点击显示 Cron 表达式说明面板，包含：
  - 格式说明：分 时 日 月 星期
  - 字段范围说明
  - 常用示例（每天 0 点、每周一 9 点、每 30 分钟等）
  - 特殊符号说明（* / , -）

**文件：** `src/components/CronManager.jsx`, `server/routes/cron.js`

---

### 7. Cron 任务编辑功能 ✅
**问题：** 创建后无法修改 Cron 任务

**修复：**
- 每个 Cron 任务卡片添加"编辑"按钮（✏️）
- 点击弹出编辑表单（可修改名称、命令、执行时间、说明）
- 保存后调用 `openclaw-cn cron update`
- 表单标题根据模式动态显示（创建/编辑）

**文件：** `src/components/CronManager.jsx`
