# 📅 Dashboard Cron 日报设置指南

## 如何设置每天晚上 0 点发送日报

### 步骤 1：打开 Dashboard
访问 `http://192.168.0.110:18790`

### 步骤 2：进入 Cron 任务
找到"Cron 任务"模块，点击 **"+ 新建"** 按钮

### 步骤 3：填写表单

#### 任务类型
选择 **"自定义任务"**

#### 任务名称
```
每日日报提醒
```

#### 执行时间
选择 **"每天 0 点 (0 0 * * *)"**

#### 执行命令（多行输入框）
```
--event "请生成 Rocky 的今日日报"

日报格式：
【🦞 OpenClaw 日报】{日期}

一、今日完成
- 任务 1
- 任务 2

二、遇到的问题
- 问题 1
- 问题 2

三、明日计划
- 计划 1
- 计划 2

四、其他说明
（可选）
```

#### 任务说明（可选）
```
每天午夜 0 点自动生成并发送日报到飞书
```

#### 启用开关
✅ 勾选"创建后立即启用"

### 步骤 4：点击"创建任务"

完成！每天 0 点你会收到飞书消息，包含当天的日报。

## Cron 表达式说明

### 格式
```
分 时 日 月 周
```

### 常用示例

| 表达式 | 含义 | 中文描述 |
|--------|------|----------|
| `0 0 * * *` | 每天 0 点 | 每天午夜 |
| `0 8 * * *` | 每天 8 点 | 每天早上 8 点 |
| `0 9 * * 1` | 每周一 9 点 | 周一早上 9 点 |
| `0 0 * * 0` | 每周日 0 点 | 周日午夜 |
| `0 0 1 * *` | 每月 1 号 0 点 | 每月第一天 |
| `*/30 * * * *` | 每 30 分钟 | 半小时一次 |
| `0 */2 * * *` | 每 2 小时 | 两小时一次 |

### 字段说明

| 字段 | 范围 | 说明 |
|------|------|------|
| 分 | 0-59 | 分钟 |
| 时 | 0-23 | 小时（24 小时制） |
| 日 | 1-31 | 日期 |
| 月 | 1-12 | 月份 |
| 周 | 0-6 | 星期（0=周日，1=周一...6=周六） |

### 特殊符号

| 符号 | 含义 | 示例 |
|------|------|------|
| `*` | 任意值 | `* * * * *` = 每分钟 |
| `/` | 间隔 | `*/15 * * * *` = 每 15 分钟 |
| `,` | 枚举 | `0 9,17 * * *` = 每天 9 点和 17 点 |
| `-` | 范围 | `0 9-17 * * 1-5` = 工作日 9 点到 17 点 |

## 日报格式自定义

你可以完全自定义日报格式，写在"执行命令"多行输入框中：

### 简洁版
```
--event "生成今日日报：完成的任务、遇到的问题、明日计划"
```

### 详细版
```
--event "请生成 Rocky 的日报"

请包含以下内容：
1. 【工作进展】今天完成了哪些任务
2. 【问题记录】遇到了什么困难，如何解决的
3. 【明日规划】明天计划做什么
4. 【资源需求】是否需要协助或资源
5. 【心情指数】用 1-5 星表示今天的心情

语气：专业但轻松，带点幽默感
```

### 项目追踪版
```
--event "生成项目日报"

项目日报格式：
【项目名称】XXX
【当前阶段】开发中/测试中/已上线
【今日进度】具体完成的功能
【阻塞问题】需要协调的资源
【风险评估】低/中/高
【明日目标】下一步计划
```

## 测试 Cron 任务

创建任务后，可以立即测试：

1. 在 Cron 任务列表中找到刚创建的任务
2. 点击"▶️ 运行"按钮（如果有）
3. 或等待下一次执行时间

## 常见问题

### Q: 命令中的 `--event` 是什么意思？
A: 这是告诉 OpenClaw 这是一个系统事件，会在主会话中触发消息。

### Q: 可以设置多个日报吗？
A: 可以！创建多个 Cron 任务，设置不同时间即可。

### Q: 如何暂停日报？
A: 在 Cron 任务列表中找到任务，关闭启用开关即可。

### Q: 如何删除日报？
A: 点击任务卡片上的"🗑️ 删除"按钮。

### Q: 如何修改日报格式？
A: 点击任务卡片上的"✏️ 编辑"按钮，修改命令内容后保存。
