## 🎯 项目概述

为 OpenClaw-CN 打造一个**轻量级、高颜值、实时**的状态监控面板，让用户随时了解 AI 助手的工作状态、任务进度和系统健康度。

### 产品定位

| 维度 | 定位 |
|------|------|
| **目标用户** | OpenClaw 个人用户、小型团队 |
| **核心价值** | 无需终端命令，可视化监控 AI 状态 |
| **差异化** | 轻量（<100MB）、美观、开箱即用 |
| **使用场景** | 日常监控、任务追踪、故障排查 |

## 📊 功能需求

### P0 - 核心功能（MVP 必须）

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|----------|
| F1 | 网关状态监控 | 显示网关运行状态、端口、启动时间、PID | 状态准确，实时更新 |
| F2 | 实时任务状态 | 显示当前任务描述、任务类型、开始时间 | 任务变化<1 秒内更新 |
| F3 | 任务计时器 | 显示任务运行时长，>5 分钟警告，>10 分钟告警 | 计时准确，警告明显 |
| F4 | 响应式设计 | PC、平板、手机完美适配 | 主流设备测试通过 |
| F5 | 深色/浅色模式 | 自动跟随系统，可手动切换 | 切换流畅，无闪烁 |

### P1 - 重要功能（v1.0 包含）

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|----------|
| F6 | TODO 列表 | 显示规划中的任务队列 | 与 AI 记忆同步 |
| F7 | MD 文档在线编辑 | 编辑 SOUL.md, USER.md, MEMORY.md 等 | 保存成功，实时生效 |
| F8 | Cron 任务管理 | 可视化查看、添加、编辑 cron 任务 | CRUD 操作完整 |
| F9 | 快捷命令面板 | 常用 openclaw-cn 命令快捷按钮 | 命令执行成功 |

### P2 - 进阶功能（v1.1+）

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|----------|
| F10 | 资源监控 | CPU、内存使用率图表 | 数据准确，刷新流畅 |
| F11 | 日志实时流 | WebSocket 推送实时日志 | 延迟<500ms |
| F12 | 会话历史查看 | 查看最近的对话历史 | 加载快速，分页正常 |
| F13 | 公网访问 | 支持外部网络访问（可选） | 配置简单，连接稳定 |
| F14 | 主题定制 | 自定义配色方案 | 即时预览，保存生效 |

## 🔐 安全设计

### v1.0（内网模式）

| 措施 | 实现 |
|------|------|
| 本地绑定 | 默认 bind=127.0.0.1 |
| 命令白名单 | 只允许预定义命令 |
| 二次确认 | 危险操作需确认 |
| 输入过滤 | 防止命令注入 |

### v2.0（公网模式）

| 措施 | 实现 |
|------|------|
| Token 认证 | JWT 验证 |
| HTTPS | TLS 加密 |
| IP 白名单 | 可配置允许列表 |
| 速率限制 | 防止暴力攻击 |
| 审计日志 | 记录所有操作 |

## 🚀 部署方案

### 开发模式

```bash
cd openclaw-dashboard
npm install
npm run dev
# 访问 http://localhost:5173
```

### 生产模式

```bash
npm run build
npm run start
# 访问 http://localhost:18790
```

### 开机自启（可选）

```bash
# 创建 launchd 服务
cp com.clawdbot.dashboard.plist ~/Library/LaunchAgents/
```
