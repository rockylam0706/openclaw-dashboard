## 🐛 已知问题排查

### 问题 1：Markdown 不渲染

**症状**：消息显示为纯文本，无格式

**可能原因**：
- marked 或 DOMPurify 未安装
- 浏览器缓存旧版 JS 文件

**解决方案**：
```bash
# 检查依赖
npm list marked dompurify

# 重新安装
npm install marked dompurify

# 重新构建
npm run build

# 重启服务
npx pm2 restart openclaw-dashboard

# 清除浏览器缓存（Ctrl+Shift+R）
```

### 问题 2：复制按钮不显示

**症状**：鼠标悬停无复制按钮

**可能原因**：
- CSS 未加载
- 浏览器不支持 `:hover` 伪类

**解决方案**：
```bash
# 检查 CSS 文件
ls -la dist/assets/*.css

# 清除浏览器缓存
# 硬刷新：Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac)
```

### 问题 3：工具消息未折叠

**症状**：工具消息完整显示，无摘要栏

**可能原因**：
- 后端未正确识别工具消息
- `toolInfo` 提取失败

**排查步骤**：
1. 打开浏览器开发者工具
2. 查看 Network 标签
3. 点击 `/api/chat/webui-messages` 请求
4. 查看 Response 中的 `toolMessages` 数组
5. 检查 `toolName` 和 `toolStatus` 字段是否存在
